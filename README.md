# Data Engineering Introduction: Azure Synapse Analytic Studio Usage
This introduction explains the basic usage of Azure Synapse Analytics Studio

---

## What is Azure Synapse Analytics?

Azure Synapse Analytics is a unified platform for ingesting, managing, and analyzing data. It can pull data from multiple sources, including cloud, multi-cloud, and on-premises systems, and process it to create dashboards and visual graphs that make insights easy to understand.

By consolidating data workflows into a single platform, it reduces the complexity of using multiple tools. The platform is browser-based, so there is no need to install additional software or applications.

Azure Synapse offers flexible pricing models, with the standout being the serverless model, which charges based on the amount of data processed—similar to Google BigQuery.

---

## Key Features of Azure Synapse Analytics

- Modern SQL-based Engine – Built on a next-generation SQL engine, supporting a wide range of processing scenarios.

- Code-friendly Processing – Allows users who prefer coding over SQL to write custom code for data processing.

- Code-free Data Transformation – Supports Data Flows for transforming data without writing code.

- Seamless Power BI Integration – Enables creating reports and dashboards directly from data within Synapse Studio.

- Synapse Link for Cosmos DB – Instantly analyze data from Azure Cosmos DB without complex ETL.

- Data Masking – Protects sensitive customer information by masking data for users without proper access rights.
